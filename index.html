GitHub Presenter

非公開GitHubリポジトリを、見せられるカタチに。

PATの作り方 / このツールについて
  1. GitHubの Settings → Developer settings → Personal access tokens でトークンを発行してください。
  2. 非公開リポジトリを読む場合は repo スコープ（Fine-grained の場合は対象リポジトリの Contents: Read-only）が必要です。
  3. トークンはこの端末のブラウザ内（localStorage）にのみ保存され、外部サーバーには送信されません。共有PCでは「保存する」のチェックを外してください。
